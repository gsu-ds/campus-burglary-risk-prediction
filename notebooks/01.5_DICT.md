## üìã Processed Data - Dictionary

This table documents the structure and source of the key features in the final processed dataset (`target_crimes.parquet`/`.csv`).

| Feature Name | Data Type (Recommended) | Source/Transformation | Description |
| :--- | :--- | :--- | :--- |
| `incident_number` | `object` | Raw APD Data | Unique identifier for the crime report. Used for deduplication. |
| `report_date` | `datetime64[ns]` | Raw APD Data | Robustly parsed incident date and time. |
| `latitude` / `longitude` | `float64` | Raw APD Data | Geographic coordinates of the incident. |
| `nibrs_offense` | `object` | Raw APD Data | Original NIBRS offense description (cleaned and lowercased). |
| `npu` | `category` | Geospatial Join (Shapefile) | Neighborhood Planning Unit (A-Z or **OUT_OF_NPU_BOUNDS**). |
| `zone_int` | `int64` | Geospatial Join (Shapefile) | APD Police Zone (1-6), imputed using NPU mode. |
| `is_holiday` | `int8` | Temporal Feature | Binary flag (1=True) if incident occurred on a US/GA holiday. |
| `hour_block` | `category` | Temporal Feature | Categorical 4-hour time bin of the incident. |
| `hour_sin` / `hour_cos` | `float64` | Temporal Feature | Cyclic encoding of the hour (0-23). |
| `temp_f` | `float64` | Open-Meteo API | Hourly ambient temperature in Fahrenheit. |
| `is_raining` | `int8` | Weather Flag | Binary flag (1=True) if precipitation $> 0.01$ inches during the hour. |
| `campus_label` | `category` | Closest Distance | Label of the nearest university campus within 1.5 miles. |
| `campus_distance_m` | `float64` | Vectorized Haversine | Distance to the nearest university center in meters. |
| `crime_density_7d_1mi` | `int64` | Spatio-Temporal Lag (Future) | Count of other target crimes in the past 7 days within a 1-mile radius. |
| `loc_type_count` | `int64` | Count Encoding | Global frequency of the location type (e.g., how many crimes occurred at 'parking lot'). |
| `npu_crime_avg_30d` | `float64` | Temporal Lag | Lagged 30-day average of daily crime count within the NPU. |
